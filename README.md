# Mbita Emmanuel — Personal Website

A full-featured professor personal website built with Next.js 14, Prisma, and PostgreSQL.

## Features
- Public website with home, about, research, publications, teaching, blog, events, gallery
- Admin dashboard with full CRUD for all content
- Academic profile links (Google Scholar, ORCID, ResearchGate, etc.)
- Contact form, announcements, testimonials
- LMS features: courses, attendance, grades, forum

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Database**: PostgreSQL + Prisma ORM
- **Styling**: Tailwind CSS
- **Auth**: iron-session + bcryptjs
